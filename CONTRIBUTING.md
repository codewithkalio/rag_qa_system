# Contributing

Thank you for your interest in this project.

At this time, this repository is **not accepting external contributions**.

Please do not open pull requests for code changes, features, or documentation updates. Unsolicited pull requests may be closed without review.

Issues and feature requests may also be closed if they are outside the current scope of maintenance.

This policy helps keep the project maintainable and aligned with the maintainer's current goals and capacity.

The maintainer may continue to make direct updates to this repository.

Thank you for understanding.
